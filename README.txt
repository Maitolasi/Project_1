This project is a simple Mood and Activity Tracker built using HTML, CSS, and vanilla JavaScript. The purpose of the application is to allow users to record their daily mood and a short activity, and then view their previous entries.

The user can select a mood using buttons, write an activity, and save the entry. Each entry is displayed in a list and can be deleted individually. The application also includes filtering by mood and a search function to find specific entries. It also shows basic statistics, such as the total number of entries.

The app uses multiple event listeners, including click events for buttons, a submit event for the form, input events for search, and change events for filtering. DOM manipulation is used to dynamically create, update, and remove elements from the page.

All data is stored in the browser using localStorage, so the entries remain saved even after refreshing the page.

The layout is simple, responsive and works on both desktop and mobile using very simple CSS and media queries.

This project helped me practice JavaScript fundamentals such as event handling, working with arrays and objects, and updating the DOM dynamically.
